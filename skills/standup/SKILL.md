# Standup

## When to activate
- Start of session
- User asks what's on, catch me up, standup

## Steps

1. Read `MEMORY.md` — last 2–3 dated sections.
2. Skim `output/daily/` for the last 3 days (filenames only unless user asks for detail).
3. Read `context/about-me.md` for current priorities.
4. Return a short brief:
   - **Carry forward** — open loops from MEMORY.md
   - **Recent output** — anything worth revisiting
   - **Today's focus** — 1–3 suggested priorities (ask if unclear)

Keep it scannable. Bullets, not paragraphs.

## What this skill does NOT do
- Does not run email or calendar unless user asks
- Does not write files unless user then asks for a plan to be saved
