# Scheduled tasks (example workspace)

Cowork does not read these files automatically. Each `.task.md` is documentation plus the prompt to paste when creating a schedule in Cowork.

After `./scripts/install.sh`, recreate enabled tasks via Cowork → `/schedule` or the scheduled tasks UI.

| File | Skill | Schedule |
|------|-------|----------|
| weekly-review.task.md | /weekly-review | Fri 4pm |

See [spec.md](../../../docs/product/spec.md) for the full `.task.md` format.
